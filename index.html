Slidecalendartimepicker

SlideCalendarTimePicker is a combine Calendar and TimePicker in one View(Dialog).

View project on GitHub

This Library is base on SlideDateTimePicker and android-times-square.

Setup

dependencies { compile project(':library') }

How to Use

final SlideCalendarTimePicker picker = new SlideCalendarTimePicker.Builder(getSupportFragmentManager()) .setInitialDate(new Date()) .setMaxDate(maxDate) .setMinDate(minDate) .setIs24HourTime(true) .setIndicatorColor(getResources().getColor(android.R.color.holo_green_light)) .setListener(new SlideDateTimeListener() { @Override public void onDateTimeSet(Date date) { String time = format(date); Toast.makeText(getApplicationContext(), time, Toast.LENGTH_LONG).show(); } }).build(); picker.show();
